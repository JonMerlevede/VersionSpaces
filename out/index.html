Version Spaces Processor

This is an implementation of the Version Space algorithm in Javascript made using Haxe for the Comparative Programming Languages course at the University of Leuven. An explanation of this algorithm in dutch can be found here.

There are three predefined structure and sample sets. They can be loaded by clicking any of the three buttons below (Regular, Extended and Extended - difficult). Pressing the Derive button starts the Version Space algorithm. You can insert your own structures and sample sets by modifying the values in the text areas.

The three examples were taken from here. An explanation on how to derive results for them yourself can also be found there.
