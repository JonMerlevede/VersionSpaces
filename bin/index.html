Version Spaces Processor

Created by Jonathan Merlevede
Last edited: 3/11/2012

This is an implementation of the Version Space algorithm in Javascript made using Haxe for the Comparative Programming Languages course at the University of Leuven in Belgium. An explanation of this algorithm in dutch can be found here.

There are three predefined structure and sample sets. They can be loaded by clicking any of the three buttons below (Regular, Extended and Extended - difficult). Pressing the Derive button starts the Version Space algorithm. These three examples were taken from here. An explanation on how to derive results for them yourself can also be found there.

You can easily insert your own structures and sample sets by modifying the values in the text areas. The application will not necessarily give a meaningful error message when presented with incorrectly formatted input.
